# Installation

```
localhost:/opt/pleroma# sudo -Hu pleroma MIX_ENV=prod mix pleroma.emoji ls-packs -m https://raw.githubusercontent.com/david-batranu/pleroma-emoji-yahoo-messenger/master/index.json
```

```
localhost:/opt/pleroma# sudo -Hu pleroma MIX_ENV=prod mix pleroma.emoji get-packs -m https://raw.githubusercontent.com/david-batranu/pleroma-emoji-yahoo-messenger/master/index.json ym
```
